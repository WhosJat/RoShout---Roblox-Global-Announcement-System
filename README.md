# RoShout---Roblox-Global-Announcement-System
A Roblox Global Announcement System with lots of customizability and no limits

GLOBAL ANNOUNCEMENT SYSTEM
FULLY CUSTOMIZABLE!

TO USE, CHANGE THE SETTINGS TO ALLOW YOUR GROUP/OR USER TO YOURS! Found in ServerScriptStorage.Cmdr.Hooks.BeforeRun

FOR REPLICATEDSTORAGE, UNGROUP THE FOLDER CONTAINING THE OTHER FOLDERS

This service uses an abridged version of Cmdr.
If your game already uses Cmdr, you can make our systems work directly with your existing Cmdr setup.

PRESS F2 TO USE AND TYPE. NO OTHER WAY TO DO IT AS OF NOW, BUT YOU CAN CONFIGURE IT TO!
Ex. announce Hello, everyone!
